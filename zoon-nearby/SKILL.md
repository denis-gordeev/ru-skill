# Zoon.ru Nearby Search

Поиск ближайших заведений и организаций через публичные страницы Zoon.ru.

## Что это такое

`zoon-nearby` — это дополнительный источник для поиска заведений (рестораны, кафе, бары) в российских городах через серверно отрендеренные страницы Zoon.ru. Этот навык дополняет `osm-nearby` (Overpass API), предоставляя более подробную информацию: рейтинги, телефоны, режим работы, описания.

## Когда использовать

- **Используй `osm-nearby`**, когда нужны только координаты и базовые данные заведений (free, no-key, fastest)
- **Используй `zoon-nearby`**, когда нужны рейтинги, отзывы, телефоны, режим работы и другие детали организаций

## Установка

```bash
npm install zoon-nearby
```

## Ключевые функции

- Поиск по городу и категории (рестораны, кафе, бары, отели, аптеки и т.д.)
- Общий текстовый поиск по Zoon.ru
- Получение детальной карточки организации
- Пагинация для больших результатов

## Примеры вызова

```
Найди рестораны в Москве
Покажи кафе в Санкт-Петербурге
Какие бары есть в Казани?
Дай подробную информацию о ресторане по ссылке https://zoon.ru/msk/restaurants/pushkin
```

## API

### `searchRestaurants(city, opts?)`

Поиск ресторантов в городе по названию города.

### `searchCafes(city, opts?)`

Поиск кафе в городе по названию города.

### `searchBars(city, opts?)`

Поиск баров в городе по названию города.

### `searchByCategory(city, category, opts?)`

Поиск по произвольной категории.

### `search(query, city?, opts?)`

Общий текстовый поиск по Zoon.ru.

### `getBusinessDetails(businessUrl, opts?)`

Получение детальной карточки заведения по URL страницы Zoon.ru.

## Что возвращается

Для поиска:
- Название организации
- Адрес
- Рейтинг
- Телефон
- Категория
- Ссылка на страницу

Для детальной карточки:
- Полное название
- Полный адрес
- Рейтинг
- Телефон
- Описание
- Режим работы
- Официальный сайт

## Технические детали

- **Источник**: Zoon.ru (публичные SSR-страницы)
- **API keys**: Не требуются
- **Anti-bot**: Отсутствует
- **Метод**: Парсинг HTML с серверно отрендеренных страниц
- **Лимиты**: Разумное использование, кэширование при частых запросах
