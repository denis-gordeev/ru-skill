# Поиск товаров Daiso

## Что делает этот навык

Проверяет **наличие товаров в конкретных магазинах Daiso** через официальные страницы поиска/магазинов/остатков Daiso Mall.

- Находит код магазина через официальный поиск магазинов.
- Находит кандидатов-товаров через официальный поиск товаров.
- Проверяет остатки в конкретном магазине через официальную страницу остатков самовывоза.
- **Если официальные страницы не указывают расположение внутри магазина, отвечает только по остаткам.**

## Когда использовать

- "Есть ли Ridley Shot в магазине Daiso на 강남?"
- "Проверь остатки в конкретном магазине Daiso"
- "Проверь, в каких магазинах есть этот товар"
- "Проверь остатки в магазине Daiso по названию"

## Когда не использовать

- Когда нет ни названия магазина, ни названия товара для однозначного ответа
- Когда нужно автоматизировать оплату/заказ/бронирование самовывоза
- Когда нужно использовать неофициальные результаты парсинга

## Предварительные требования

- Подключение к интернету
- `node` 18+
- Пакет `daiso-product-search` этого репозитория или эквивалентная логика

## Необходимые входные данные

### 1. Сначала запросите название магазина, если оно отсутствует

Без названия магазина не начинайте поиск, сначала спросите.

- Рекомендуемый вопрос: `Какой магазин Daiso проверить? Укажите название магазина (например: 강남역2호점).`
- Если найдено несколько похожих магазинов: `Найдено несколько подходящих магазинов. Выберите один из списка.`

### 2. Запросите название товара или ключевое слово, если оно отсутствует

Название товара или поисковое ключевое слово также обязательно.

- Рекомендуемый вопрос: `Укажите название товара или ключевое слово для поиска. Например: VT 리들샷 100`
- Если запрос слишком широкий: `Поисковый запрос слишком общий. Укажите бренд или объём.`

## Официальные страницы Daiso Mall

- каталог магазинов по ключевому слову: `https://www.daisomall.co.kr/api/ms/msg/selStrSrchKeyword`
- поиск магазинов: `https://www.daisomall.co.kr/api/ms/msg/selStr`
- детали магазина: `https://www.daisomall.co.kr/api/dl/dla-api/selStrInfo`
- краткая информация о товаре: `https://www.daisomall.co.kr/ssn/search/Search`
- список товаров: `https://www.daisomall.co.kr/ssn/search/SearchGoods`
- краткий список товаров: `https://www.daisomall.co.kr/ssn/search/GoodsMummResult`
- остатки самовывоза в магазине: `https://www.daisomall.co.kr/api/pd/pdh/selStrPkupStck`
- опциональная перекрёстная проверка онлайн-остатков: `https://www.daisomall.co.kr/api/pdo/selOnlStck`

## Рабочий процесс

### 1. Определение магазина

Сначала разрешите название магазина через официальный API поиска магазинов.

```js
const { searchStores } = require("daiso-product-search")

const storeResult = await searchStores("강남역2호점", {
  limit: 5
})

console.log(storeResult.items)
```

Если найдено несколько кандидатов, покажите топ-2 или 3 и запросите подтверждение.

### 2. Определение товара

Найдите кандидатов-товаров через официальную страницу `SearchGoods`.

```js
const { searchProducts } = require("daiso-product-search")

const productResult = await searchProducts("VT 리들샷 100", {
  limit: 10
})

console.log(productResult.items)
```

Если найдено несколько кандидатов, кратко отфильтруйте по приоритету:

- Точное совпадение названия
- Название с брендом и объёмом/номером
- Кандидаты с высоким числом отзывов/поисковым рейтингом
- При необходимости перекрёстной проверки онлайн-остатков сохраните `onldPdNo` кандидата

### 3. Проверка остатков самовывоза в магазине

Проверьте остатки в конкретном магазине через официальный API.

```js
const { getStorePickupStock } = require("daiso-product-search")

const stock = await getStorePickupStock({
  pdNo: "1049275",
  strCd: "10224"
})

console.log(stock)
```

### 4. Используйте сквозной helper, когда оба названия уже известны

```js
const { lookupStoreProductAvailability } = require("daiso-product-search")

const result = await lookupStoreProductAvailability({
  storeQuery: "강남역2호점",
  productQuery: "VT 리들샷 100"
})

console.log(result.selectedStore)
console.log(result.selectedProduct)
console.log(result.pickupStock)
```

### 5. Консервативный ответ

Ответ должен быть кратким и чётким.

- Название магазина
- Название товара
- Количество в магазине или отсутствие остатков
- При необходимости укажите ссылку на онлайн-остатки
- **Если официальные страницы не указывают расположение внутри магазина, явно сообщите: «Официальные страницы показывают только остатки в магазине».**

## Считается выполненным, когда

- Название магазина и товара оба определены.
- Кандидаты магазинов и товаров найдены через официальные страницы.
- Результат остатков в магазине возвращён хотя бы один раз.
- Если информация о расположении отсутствует, явно сообщите об этом.

## Режимы сбоев

- При слишком широком названии магазина может быть найдено несколько точек одного района.
- При слишком широком названии товара может быть много кандидатов с разным объёмом/номером.
- Официальные остатки могут отличаться от фактических на момент визита.
- Текущие официальные страницы могут **не указывать aisle/расположение внутри магазина**.

## Примечания

- Это навык для запросов.
- Сохраняйте приоритет официальных страниц.
- Если официальные страницы не указывают расположение, не делайте предположений.
