# Гайд по поиску баров рядом через Kakao Map

## Что умеет этот сценарий

- Превращать запросы вроде `Сеульский вокзал`, `Gangnam`, `Sadang` или `Nonhyeon` в поиск баров по Kakao Map.
- Сначала поднимать заведения, которые сейчас открыты.
- Возвращать меню, варианты посадки (`компания`, `барная стойка` и т.п.) и номер телефона.
- Примерно оценивать расстояние между опорной точкой и найденными барами.

## Что спросить первым

Этот сценарий нужно запускать только после уточнения текущего местоположения пользователя (`현재 위치`).

Рекомендуемый вопрос:

```text
Напишите, где вы сейчас находитесь. Подойдут название станции, района или ориентира вроде Seoul Station, Gangnam или Sadang. После этого я найду ближайшие бары по Kakao Map.
```

## Входные данные

- Название станции: `서울역`, `사당`, `강남`, `신논현`, `논현`
- Район или ориентир: `해방촌`, `코엑스`, `성수동`

Если зона слишком широкая или расплывчатая, лучше сузить её до ближайшей станции.

## Официальные поверхности Kakao Map

- Мобильный поиск: `https://m.map.kakao.com/actions/searchView?q=<query>`
- JSON панели места: `https://place-api.map.kakao.com/places/panel3/<confirmId>`
- Страница места: `https://place.map.kakao.com/<confirmId>`

Базовый поток такой: `запрос по месту` -> `выбор anchor-точки` -> `поиск "место + бар"` -> `нормализация panel3`.

## Какие поля нормализуются

- Название бара и категория
- Текущий статус работы (`영업 중`, `영업 전`, `휴무일`)
- Примеры `메뉴`
- Посадка и ориентиры по вместимости (`단체석`, `바테이블` и т.п.), то есть `좌석 옵션` и `인원 수용` подсказки
- Телефон (`전화번호`)
- Расстояние, если его можно оценить

## Пример на Node.js

```js
const { searchNearbyBarsByLocationQuery } = require("kakao-bar-nearby");

async function main() {
  const result = await searchNearbyBarsByLocationQuery("서울역", {
    limit: 5
  });

  console.log(result.anchor);
  console.log(result.items);
}

main().catch((error) => {
  console.error(error);
  process.exitCode = 1;
});
```

## Практические советы

- Если есть открытые бары, показывайте их первыми; если нет, добавляйте те, что скоро откроются.
- Если меню пустое, опирайтесь на категорию Kakao Map и краткое описание, чтобы дать грубую оценку кухни.
- По ключевым словам вроде `단체석`, `룸`, `바테이블` можно приблизительно объяснить, подходит ли место для компании.
- Ограничивайтесь справочной выдачей, без перехода к бронированию или заказу.

## Ограничения

- `panel3` JSON может отвечать `406`, если запрос не похож на браузерный.
- Если Kakao Map изменит структуру панели места, поля меню и статуса работы тоже могут сместиться.
- Точная вместимость не всегда доступна, поэтому лучше формулировать вывод как приблизительный ориентир.

Историческое имя сценария для doc-регрессий: `근처 술집 조회`.
