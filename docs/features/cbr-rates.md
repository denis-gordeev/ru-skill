# Курсы валют Банка России

## Что делает этот навык

- Получает официальный курс валют ЦБ РФ из публичного XML-сервиса
- Поддерживает запрошенную дату в формате `YYYY-MM-DD`
- Возвращает предыдущее доступное опубликованное значение и изменение к нему

## Что нужно заранее

- Node.js 18+
- После публикации: `npm install -g cbr-rates`
- Перед запуском: `export NODE_PATH="$(npm root -g)"`
- При разработке в этом репозитории: `npm install`

## Входные данные

- Код валюты: `USD`, `EUR`, `CNY`
- Необязательная дата

## Базовый сценарий

1. Если пакет не установлен, сначала поставить `cbr-rates`, а не обходить это ручным XML-парсером.
2. Запросить официальный курс по коду валюты и дате.
3. При необходимости добавить предыдущее доступное опубликованное значение и изменение к нему.

## Пример

```bash
NODE_PATH="$(npm root -g)" node - <<'JS'
const { getRateWithChange } = require("cbr-rates");
getRateWithChange("USD", "2026-04-02").then((result) => console.log(JSON.stringify(result, null, 2)));
JS
```

## На что смотреть в ответе

- `requestedDate`: дата, которую запросил пользователь
- `publishedDate`: дата публикации, которую реально вернул сервис Банка России
- `unitRate`: курс за одну единицу валюты в рублях
- `change`: изменение к предыдущей доступной публикации

## Ограничения

- Это только read-only сценарий
- Источник данных: официальный XML-сервис Банка России `https://www.cbr.ru/scripts/XML_daily.asp`
