# Бары Kakao поблизости

## Что делает этот навык

Ищет **бары поблизости через Kakao Map** на основе текущего местоположения, указанного пользователем.

- Местоположение не определяется автоматически.
- **Всегда сначала спрашивайте текущее местоположение**.
- Может принимать запросы станций/районов/ориентиров, такие как `서울역`, `강남`, `사당`, `신논현`, `논현`.
- Результаты включают текущий статус работы, основное меню, варианты размещения (групповые места/барные столы и т.д.), номер телефона.

## Когда использовать

- "Найди бары рядом с Seoul Station"
- "Какие винные бары сейчас открыты в Gangnam?"
- "Покажи несколько баров рядом с Nonhyeon для 4 человек"
- "Покажи несколько изакая с номерами телефона рядом с Sadang"

## Обязательный первый вопрос

Не начинайте поиск без информации о местоположении, обязательно спросите.

- Рекомендуемый вопрос: `Укажите ваше текущее местоположение. Пришлите название станции/района, например Seoul Station/Gangnam/Sadang, и я найду бары поблизости через Kakao Map.`
- Если местоположение нечёткое: `Уточните ещё раз, пожалуйста, по ближайшей станции или названию округа.`

## Официальные поверхности Kakao Map

- Мобильный поиск: `https://m.map.kakao.com/actions/searchView?q=<query>`
- JSON панели места: `https://place-api.map.kakao.com/places/panel3/<confirmId>`
- Страница деталей места: `https://place.map.kakao.com/<confirmId>`

## Рабочий процесс

1. Обязательно спросите текущее местоположение у пользователя.
2. Интерпретируйте полученную строку местоположения как кандидат-якорь (станция/ориентир) для поиска Kakao Map.
3. Добавьте ключевое слово `бары` к той же строке местоположения для получения результатов nearby поиска баров.
4. Запросите JSON panel3 для верхних кандидатов, чтобы нормализовать текущий статус работы, меню, варианты размещения, номер телефона.
5. **Сначала покажите открытые сейчас бары**, при необходимости также покажите скоро открывающиеся.

## Ответы

Обычно кратко резюме по 3-5 местам.

- Название бара
- Категория
- Статус работы (`Открыто`, `Закрыто`, `Выходной` и т.д.)
- 2-3 основных пункта меню
- Подсказки по местам/размещению (`групповые места`, `барные столы` и т.д.)
- Номер телефона
- Расстояние (если возможно)

## Пример Node.js

```js
const { searchNearbyBarsByLocationQuery } = require("kakao-bar-nearby");

async function main() {
  const result = await searchNearbyBarsByLocationQuery("서울역", {
    limit: 5
  });

  console.log(result.anchor);
  console.log(result.items);
}

main().catch((error) => {
  console.error(error);
  process.exitCode = 1;
});
```

## Считается выполненным, когда

- Текущее местоположение пользователя подтверждено.
- Найден хотя бы один результат бара через Kakao Map или предложены причины отсутствия результатов и следующий вопрос.
- Представлено резюме со статусом работы/меню/вариантами размещения/номером телефона.
